## Dependencies

- Ruby 3.2.2
- Rails 7.0.4
- PostgreSQL

### Local development

1. Run the setup script.

   ```sh
   $ bin/setup
   ```

1. Start the development server.

   ```sh
   $ bin/rails s
   ```
